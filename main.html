      About the Visualization

      This narrative visualization follows a martini glass hybrid structure. The first part of the narrative is a journey through time where data for each year is displayed from one scene to the next (the stem of the martini glass). The final slide allows the user to interact with the data and play with different parameters - this is the cone of the martini glass structure.

      The charts used in the first part of the visualization are all pie charts in order to keep with visual consistency. The colors for each language in the pie are also kept consistent. The pie chart animates from each year to the next in order to create a smooth transition and to show the changes dynamically as the years progress. Annotations are provided on each slide to complement the pie chart and to add additional information to highlight specific data items or trends. These annotations follow a simple header plus bullet list style which is kept consistent throughout the narrative visualization. The final exploration slide only uses text and annotations to guide the user in how to explore the data.

      There are different triggers in the narrative visualization. The top part of the site is the navigation bar which starts at 2010 and progresses across the screen. This style is used to indicate that the user should explore each tab in order and it helps with showing the state of the narrative visualization. In the year-by-year slides, the user is told they can hover over pie slices for more information. This hover action shows a tooltip with the language, percentage, and value for the slice. The exploration scene uses empty checkboxes and dropdowns to trigger actions from the user. The help text also clues the user into understanding that they can hover over data points on the graph to get more information.

      The exploration page offers the user the chance to change data parameters. The list of languages on the left side of the page with the checkboxes encourages the user to select languages from the list to plot. The darkened fill on hover also is a visual effect to help the user understand that the items in the list are clickable. Once selected, the list item will turn the color of the line used to plot the language which also acts as a key for line graph. The dropdown above the line graph also shows the user that they have options to select from in terms of what data theyâ€™d like to graph.

        User Interface Events
      • Nav bar tabs: Shows order of the narrative visualization and provides a way for a user to go from slide to slide.
      • Pie chart hover: Shows user more information about the data for the pie slice.
      • Language list in the explore scene: Clickable items that a user can use to plot different languages in the adjacent graph.
      • Data source dropdown in the explore scene: A list of options of data sources that the user can graph for the selected languages.
      • Hover on data points in the explore scene: Shows a tooltip with more information on the value of the data for that year.
